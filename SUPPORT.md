# Support

Need help or want to ask a question?

## Best place to start

- Use GitHub Discussions in this repository for general Q&A and community help.

## When to open an issue

- Use Issues for specific, actionable bugs or repository maintenance tasks.

## Security concerns

- Please do **not** report security vulnerabilities in public Discussions or
  Issues. See [SECURITY.md](./SECURITY.md) for responsible disclosure guidance.
