# Probe-Tech Community

This repository is the home for organization-wide community guidance and
discussion routing.

## What this repo is for

- Central place for community expectations and contributor guidance
- Default support/security/community docs for Probe-Tech repositories
- A simple landing page for members and external contributors

## Recommended GitHub setup (one-time)

1. In repository **Settings → General → Features**, enable **Discussions**.
2. In **Settings → Discussions**, create categories such as:
   - `Announcements`
   - `Ideas`
   - `Q&A`
   - `Show and tell`
3. Pin this repository in the organization so members can discover it quickly.
4. Link to this repository from your organization profile and onboarding docs.
